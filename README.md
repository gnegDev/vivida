
## Описание
Приложение для оптимизации плана лечения опухолей мозга с использованием AI и ML.

## Основные возможности
Приложение предоставляет удобный интерфейс для оптимизации плана лечения опухолей мозга с помощью ИИ. Имеется возможность скачивания отчёта о выполненном анализе.

## Системные требования для запуска
* Оперативная память: от 4 ГБ (8 ГБ рекомендовано)
* Ядра ЦП: от 2 (4 рекомендовано)
* Место на диске: от 20 ГБ
* Docker
* Git

## Quick start
1. ```git clone https://github.com/gnegDev/vivida.git```
2. Скачать https://drive.google.com/file/d/1hvvvdLkFtKNuCw6LWm5ar7nnSlxJI8w7/view?usp=drive_link и распаковать в ```model/vivida/``` (должен получится путь ```model/vivida/gbm_models_output_all90_dosage_full_features```
3. ```cd vivida```
4. ```docker compose up --build```
5. Приложение будет доступно на http://localhost:5000/
<br/><br/>

## Структура проекта
* ```interface/```: веб-приложение Flask на Python, предоставляющее веб-интерфейс
* ```service/```: веб-сервис Spring Boot на
Java, исполняющий обработку данных и управляющий БД PostgreSQ
* ```model/```: модель ИИ на Python, использующая TensorFlow и выполняющая основной анализ плана лечения
