## Quick start
1. Скачать https://drive.google.com/file/d/1hvvvdLkFtKNuCw6LWm5ar7nnSlxJI8w7/view?usp=drive_link и распаковать в ```model/vivida/``` (должен получится путь ```model/vivida/gbm_models_output_all90_dosage_full_features```
2. ```git clone https://github.com/gnegDev/scanity.git```
3. ```cd scanity```
4. ```docker compose up --build```
5. Приложение будет доступно на http://localhost:5000/
